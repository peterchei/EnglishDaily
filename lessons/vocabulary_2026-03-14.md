# Daily English Lesson - 2026-03-14

Hi there,

## 1. **Cheers** (Exclamation/Noun)
**Pronunciation:** /tʃɪəz/
**Definition:** An informal expression of goodwill, gratitude, or a toast.
**Cantonese Meaning:** 多謝 (do1 ze6) / 乾杯 (gon1 bui1)

**Example 1:**
"Cheers, that was a great help!"
**Cantonese Example 1:**
"多謝你，你幫咗我好多！" (do1 ze6 nei5, nei5 bong1 zo2 ngo5 hou2 do1!)

**Example 2:**
"Let's have a cheers to a successful project!"
**Cantonese Example 2:**
"我哋嚟乾杯，慶祝項目成功！" (ngo5 dei6 lei4 gon1 bui1, hing3 zuk1 hong6 muk6 sing4 gung1!)

## 2. **Queue** (Noun/Verb)
**Pronunciation:** /kjuː/
**Definition:** A line of people waiting for something. To stand in a line.
**Cantonese Meaning:** 隊 (deoi2) / 排隊 (paai4 deoi2)

**Example 1:**
There's a long queue at the bus stop.
**Cantonese Example 1:**
巴士站有條好長嘅隊。 (baa1 si2 zaam6 jau5 tiu4 hou2 coeng4 ge3 deoi2.)

**Example 2:**
You need to queue up to buy tickets.
**Cantonese Example 2:**
你要排隊買飛。 (nei5 jiu3 paai4 deoi2 maai5 fei1.)

## 3. **Brilliant** (Adjective)
**Pronunciation:** /ˈbrɪlɪənt/
**Definition:** Very intelligent; exceptionally clever or talented. Also, very good.
**Cantonese Meaning:** 叻 (lek1) / 好掂 (hou2 dim3) / 好棒 (hou2 paang5)

**Example 1:**
She's a brilliant scientist.
**Cantonese Example 1:**
佢係一個好叻嘅科學家。 (keoi5 hai6 jat1 go3 hou2 lek1 ge3 fo1 hok6 gaa1.)

**Example 2:**
That's a brilliant idea!
**Cantonese Example 2:**
呢個主意真係好掂！ (ni1 go3 zyu2 ji3 zan1 hai6 hou2 dim3!)

## 4. **Takeaway** (Noun)
**Pronunciation:** /ˈteɪkəˌweɪ/
**Definition:** A restaurant that sells cooked food to be eaten elsewhere. Also, the main point or message to be learned from something.
**Cantonese Meaning:** 外賣 (ngoi6 maai6) / 重點 (zung6 dim2)

**Example 1:**
Let's get some Chinese takeaway tonight.
**Cantonese Example 1:**
我哋今晚叫啲中餐外賣啦。 (ngo5 dei6 gam1 maan1 giu3 di1 zung1 caan1 ngoi6 maai6 laa1.)

**Example 2:**
The main takeaway from the meeting was that we need to improve our communication.
**Cantonese Example 2:**
個會議嘅重點係我哋需要改善溝通。 (go3 wui2 ji5 ge3 zung6 dim2 hai6 ngo5 dei6 seoi1 jiu3 gaai2 sin6 gau1 tung1.)

## 5. **Lovely** (Adjective)
**Pronunciation:** /ˈlʌvli/
**Definition:** Beautiful; attractive; delightful.
**Cantonese Meaning:** 靚 (leng3) / 好正 (hou2 zeng3) / 可愛 (ho2 oi3)

**Example 1:**
What a lovely dress!
**Cantonese Example 1:**
條裙真係好靚呀！ (tiu4 kwan4 zan1 hai6 hou2 leng3 aa3!)

**Example 2:**
We had a lovely time at the seaside.
**Cantonese Example 2:**
我哋喺海邊玩得好開心。 (ngo5 dei6 hai2 hoi2 bin1 waan2 dak1 hou2 hoi1 sam1.)

## 6. **Sorted** (Adjective)
**Pronunciation:** /ˈsɔːtɪd/
**Definition:** Organized; arranged; resolved.
**Cantonese Meaning:** 搞掂 (gaau2 dim6) / 整好咗 (zing2 hou2 zo2)

**Example 1:**
"Don't worry, I've got it sorted."
**Cantonese Example 1:**
"唔使擔心，我搞掂咗啦。" (m4 sai2 daam1 sam1, ngo5 gaau2 dim6 zo2 laa1.)

**Example 2:**
The paperwork is all sorted now.
**Cantonese Example 2:**
啲文件而家全部整好咗。 (di1 man4 gin2 ji4 gaa1 cyun4 bou6 zing2 hou2 zo2.)
